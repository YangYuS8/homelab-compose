# Dockge

Dockge is used as the local visual Docker Compose management panel on `infra-docker-01`.

## Runtime host

- Host: `infra-docker-01`
- URL: `http://192.168.3.30:5001`

## Paths

- Stack path: `/data/compose/stacks/dockge`
- Data path: `/data/docker/config/dockge`
- Managed stacks path: `/data/compose/stacks`

## Notes

Dockge is not the source of truth.

Git repository is the source of truth. Dockge is only a local operation panel.
